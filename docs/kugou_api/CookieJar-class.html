CookieJar

CookieJar class

Cookie 管理器，负责存储、匹配和序列化 HTTP Cookie。

支持按域名匹配 Cookie、自动解析 Set-Cookie 响应头、 Cookie 过期检查以及序列化/反序列化以便持久化存储。
